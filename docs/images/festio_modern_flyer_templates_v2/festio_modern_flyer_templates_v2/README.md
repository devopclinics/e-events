# Festio Modern Flyer Templates v2

This pack contains 24 additional modern flyer template previews for Festio Design Studio.

Contents:
- previews_1080x1350/ — full flyer previews
- thumbnails_360x450/ — gallery thumbnails
- configs/ — JSON metadata for each template
- template_manifest.json — import manifest

These are original starter template previews designed for structured editing. Wire the JSON configs to the Festio Design Studio renderer so clients can upload photos, edit text, change colors, toggle QR codes, and export PNG/PDF.
