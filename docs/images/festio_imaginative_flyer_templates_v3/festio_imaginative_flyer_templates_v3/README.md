# Festio Imaginative Flyer Templates v3

This pack contains 30 creative/imaginative flyer template previews for Festio Design Studio.

Styles include:
- Cinematic
- Afro Luxe
- Botanical
- Futurist
- Editorial / Magazine
- Postcard
- Luxury Arches
- Neon/Future

Contents:
- previews_1080x1350/
- thumbnails_360x450/
- configs/
- template_manifest.json

These are starter template previews plus metadata. The Festio Design Studio renderer should use the JSON to let organizers upload photos, edit wording, change colors, toggle QR placement, and export PNG/PDF.
